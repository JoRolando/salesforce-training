1. What problem does Salesforce solve?
Salesforce solves the problem of disconnected systems like manual spreadsheets. It brings sales, marketing, and customer service teams together onto a single, connected platform, giving everyone a shared, real-time view of customer data so things don't fall through the cracks

2.What is CRM? 
CRM (Customer Relationship Management) is a technology and business system used by companies to manage customer information, interactions, sales, support, and communication in one centralized platform.
  It helps organizations:
  store customer data,
  track sales and leads,
  manage customer service,
  automate business processes,
  and improve customer relationships.

3.What is an Object in Salesforce? Give one example.
In Salesforce, an Object is a database table used to store specific types of data. Objects contain records and fields, similar to rows and columns in a spreadsheet.
For example, the Contact object stores customer contact information such as name, phone number, and email address.

4. Difference between Salesforce Admin and Developer.
A Salesforce Admin mainly manages and configures the Salesforce platform using low-code tools like Flow Builder, reports, dashboards, and user settings to support business processes, while a Salesforce Developer builds custom applications and advanced features using programming languages such as Apex, JavaScript, and APIs. Admins focus more on configuration and system management, whereas Developers focus on coding, integrations, and custom business logic.

5.Suggest one real-world application that can be built using Salesforce.
One real-world application that can be built using Salesforce is a College Management System that manages student records, faculty details, course enrollment, attendance, fee payments, and notifications. Salesforce can automate workflows, store data securely, generate reports, and provide portals for students and staff to access information efficiently.
