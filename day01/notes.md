# Week 1 - Day 1 Notes

## 1. What problem does Salesforce solve?

Salesforce solves the problem of disconnected systems like spreadsheets and manual processes by bringing sales, marketing, and customer service into one platform.

---

## 2. What is CRM?

CRM (Customer Relationship Management) is a technology and business system used to manage customer information, interactions, sales, support, and communication.

### CRM Helps Organizations:
- Store customer data
- Track sales and leads
- Improve customer relationships
- Automate workflows

---

## 3. What is an Object in Salesforce?

In Salesforce, an Object is a database table used to store specific types of data.

### Example:
The Contact object stores:
- Name
- Phone number
- Email address

---

## 4. Difference Between Salesforce Admin and Developer

| Salesforce Admin | Salesforce Developer |
|---|---|
| Uses low-code tools | Uses programming |
| Manages workflows | Builds applications |
| Focuses on setup | Focuses on coding |

---

## 5. Real-World Application

A College Management System can be built using Salesforce to manage:
- Students
- Faculty
- Attendance
- Courses
- Fees
